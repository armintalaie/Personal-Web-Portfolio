Snake Game

Description:

I had trouble with animations in JavaFX. I wanted to make a game in Java to become more comfortable with smooth transitions and try out making a game

Technical walkthrough:

I created multiple classes that represent the game map, the snake cell. Next I created a linked list for the snake game for a faster program. I implmeneted the game logic with prize, traps, lives and score. Next, with JavaFx, I create a grid map and that the snake moves with keyboard input and the snake's end only update, removing the need to update the map and all snake cells every time. I added a smooth transtion for snake movement instead of cell by cell movement. Finally, I added a losing screen and a main menu screen.
