Movie Recommender Web App
  • Github

Description:

I'm a major movie buff and I always don't know what movie to watch. I though of doing this program to suggest me movies to watch based on my own personal taste..

Technical walkthrough:

Logic: In Python I imported a dataset of movies and user ratings. Using pandas I applied data wrangling functionalities. I removed unneccasary data and shaped the data for user. I implmeneted the ability to search movies and rate them or get random movies to rate. Next, I analyzed the input and with Lenskit's machine learning libraries I find close neighbor matches. Finally, it outputs a list of movies that match the users erates that they haven't watched. Additionally, as the input data grows the suggestions become more accurate.

User interface:

I created and designed a website using HTML, CSS and JavaScript. I then connected the logic to the interface with Django.
